# Ejercicio 4 - API REST de Plazos Fijos

Esta API permite la creación, consulta, actualización y eliminación de Plazos Fijos.
Se encuentra desarrollada con **Next.js** y **TypeScript** para el manejo de las rutas y lógica de negocio.
A este stack se le adiciona **React**, utilizado para el desarrollo de una página web mínima que facilita la interacción con la API.
Implementa además una base de datos relacional, desplegada en **mariaDB**, permitiendo la persistencia de la información ingresada.
El despliegue está automatizado mediante *Docker*.
